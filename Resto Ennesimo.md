> [!error] Teorema
> Data una [[Serie]] $\sum_{n=1}^{\infty}a_{n}$, se essa è convergente / divergente / indeterminata, allora $\forall\ N \in \mathbb{N}$ anche $$R_{N} = \sum_{n=N+1}^{\infty} a_{n}$$ converge / diverge / è indeterminata.
> In particolare:
> - Se la serie converge allora $$\lim_{ N \to \infty } R_{N} = 0$$
> - Se la serie diverge allora $$$$

> [!info] Osservazione
> Se la [[Serie]] $\sum_{n=1}^{N}a_{n}$ converge $$R_{N} = \sum_{n=N+1}^{\infty} a_{n} = $$ $$= \sum_{n=1}^{\infty} a_{n} - \sum_{n=1}^{N} a_{n}$$ $$= S - S_{N};\ \ \forall\ N \in \mathbb{N}, N \geq 1$$

> [!warning] Definizione
> $R_{N} = \sum_{n=N+1}^{\infty} a_{n}$ si chiama resto N-esimo della [[Serie]] (convergente)
