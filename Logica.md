# Definizioni
- Proposizione
	- Enunciato non contenente variabili libere a cui si possa attribuire in maniera univoca un valore di verità
	- Indicate con le lettere latine maiuscole (P, Q, ...)
		- P: 3>2
- Predicato
	- Enunciato che contiene variabili libere, il cui valore di verità dipende dalle variabili libere. Tuttavia, per ogni assegnazione delle variabili, il predicato deve essere una proposizione
	- Indicati con lettere latine maiuscole con variabili libere tra parentesi (P(x, y, ...), Q(x, y, ...))
		- P(n): n è pari
		- P(f): f è continua
- Ipotesi
	- Una o più proposizioni o predicati P
- Tesi
	- Una o più proposizioni o predicati Q
- Teorema
	- Un asserto di cui si vuole dimostrare la verità partendo dalle ipotesi
	- Dimostrare un teorema vuol dire dimostrare P $\Rightarrow$ Q
- Dimostrazione
	- Insieme dei passaggi e deduzioni logiche da una proposizione all'altra che parte da P vera e si conclude con Q vera

# Connettivi  Logici
- $\land$ = logic and
- $\lor$ = logic or
- $\lnot$ = logic not
- $\Rightarrow$ = implica (se ... allora ...)
- $\Leftrightarrow$ = doppia implicazione (se e solo se ... allora ...)

# Quantificatori
Per assegnare un valore di verità ad un predicato è necessario "saturare" le variabili libere, in modo tale da ottenere una proposizione
- $\forall$ = per ogni
- $\exists$ = esiste
- $\exists!$ = esiste ed è unico
- $\nexists$ = non esiste

# Esempi
- $P \Leftrightarrow Q$
	- $P$ è necessaria e sufficiente per $Q$
	- $Q$ è necessaria e sufficiente per $P$
- $\forall \  x \in A\ P(x)\ \Rightarrow Q(x)$
	- $P$ è sufficiente per $Q$
	- $Q$ è necessario per $P$
