#GAL 
> [!warning] Definizione
> Sia $V$ uno [[Spazi Vettoriali|Spazio Vettoriale]]
> Un [[Prodotto Scalare]] è un'operazione che associa ad ogni coppia di vettori $\underline{v}, \underline{w} \in V$ uno scalare $<\underline{v}, \underline{w}> \in \mathbb{R}$.
> Uno [[Spazi Vettoriali|Spazio Vettoriale]] dotato di un prodotto scalare si dice **Spazio Euclideo**

> [!error] Proposizione
> Sia $V$ uno **Spazio Euclideo**
> 1. $||c\underline{v}|| = |c| \cdot||\underline{v}||$
> 2. $<\underline{0}, \underline{v}> = 0$
> 3. $||\underline{v}|| = 0 \Leftrightarrow \underline{v} = \underline{0}$

> [!error] Dimostrazione
> 1. $||c\underline{v}|| = \sqrt{ <c\underline{v}, c\underline{v}> } = \sqrt{ c<\underline{v}, c\underline{v}> } = \sqrt{ c^2<\underline{v},\underline{v}> } = c\sqrt{ <\underline{v}, \underline{v}> } = |c| \cdot ||\underline{v}||$
> 2. $<\underline{0}, \underline{v}> = <0\underline{0}, \underline{v}> = 0 <\underline{0}, \underline{v}> = 0$
> 3. segue dalla proprietà di Definitivamente Positivo di [[Prodotto Scalare]]
