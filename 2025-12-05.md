$$\lim_{ t \to \infty } \int_{-t}^t \frac{x}{1+x^2}dx = 0$$
$$ \not\exists \int_{-\infty}^\infty \frac{x}{1+x^2}dx = \int_{-\infty}^0 \frac{x}{1-x^2} dx + \int_{0}^\infty \frac{x}{1+x^2}dx = \lim_{ t \to \infty } \int_{-t}^0 \frac{x}{1+x^2} dx + \lim_{ t \to \infty } \int_{0}^t \frac{x}{1+x^2} dx = $$
$$= \lim_{ t \to \infty } \int_{-t}^0 \frac{x}{1+x^2}dx + \int_{0}^t \frac{x}{1+x^2}dx \overset{\text{Somma dei Domini}}{=} \lim_{ t \to \infty } \int_{-t}^t \frac{x}{1+x^2} dx $$
