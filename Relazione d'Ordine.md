#analisi 
- Definita  ==Relazione d'Ordine Totale== ($\leq$) in $\mathbb{R}$ e $\mathbb{Q}$
- Non definita in $\mathbb{C}$

### Proprietà
- $O_{1}$: Riflessività: $a \leq a$
- $O_{2}$: Antisimmetria: $a\leq b,b\leq a \implies a=b$
- $O_{3}$: Transitività: $a\leq b, b\leq c\implies a\leq c$
- $O_{4}$: La relazione d'ordine è totale: $\forall\ a, b: a\leq b  \lor b \leq a$
- $SO$: $\forall\ a, b, : se a\leq b\implies a+c\leq b+c$
- $PO$: $\forall a, b \land \forall c > 0, a<b \implies ac<bc$

def:
- se x è un insieme dotato di operazioni + e $\cdot$, somma e prodotto, e di una relazione d'ordine totale ($\leq$) che soddisfa:
	- $S_{1}-S_{4}$
	- $P_{1} - P_{4}$
	- $O_{1} - O_{4}$
	- $SO,\ PO,\ SP$
- allora è detto campo ordinato
	- e.g. [[Numeri Reali]], [[Numeri Razionali]]
	- [[Numeri Complessi]] è un campo ma non può essere reso campo _ordinato_
> [!info] Osservazione
> Tutte le familiari regole di calcolo che utilizziamo abitualmente in $\mathbb{R}$ ed in $\mathbb{Q}$ possono essere dedotte da questi assiomi
