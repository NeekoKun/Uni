$\dfrac{dv}{dt} = a_{t_{0}} \implies v(t) = v_{0} + \int_{t_{0}}^ta_{t_{0}}dt = v_{0} + a_{t_{0}}(t-t_{0})$

$\dfrac{d\omega}{dt} = \alpha_{0} \implies \omega(t) = \omega_{0} + \int_{t_{0}}^t\alpha_{0}dt = \omega_{0} + \alpha_{0}(t-t_{0})$

$s(t) = s_{0} + \int_{t_{0}}^t v(t)dt = \boxed{s_{0} +v_{0}(t-t_{0}) + \frac{1}{2}a_{t_{0}}(t-t_{0})^2}$
$\theta(t) = \theta_{0} + \int_{t_{0}}^t \omega(t)dt = \boxed{s_{0} +\omega_{0}(t-t_{0}) + \frac{1}{2}\alpha_{0}(t-t_{0})^2}$
