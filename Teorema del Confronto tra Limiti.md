#analisi 
> [!error] Teorema
> Se $x_{0} \in \overline{\mathbb{R}}$, $f,g,h: I \subseteq \mathbb{R} \to \mathbb{R}, x_{0}$ punto di accumulazione per $I$, $l \in \mathbb{R}$
> Se $\exists \lim_{ x \to x_{0} } f(x) = \lim_{ x \to x_{0} } h(x) = l$
> ed inoltre definitivamente per $x \to x_{0}$ vale $$f(x) \leq g(x) \leq h(x)$$ allora $\exists\ \lim_{ x \to x_{0} } g(x) = l$
