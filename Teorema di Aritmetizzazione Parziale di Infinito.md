#analisi 
> [!error] Teorema
> Siano $a_{n}, b_{n}$
> 1. Se $\lim_{ n \to \infty }a_{n}=a\in \mathbb{R}, \lim_{ n \to \infty }b_{n}=\pm \infty$$$\implies \lim_{ n \to \infty }a_{n}+b_{n} = \pm \infty$$
> 2. Se $\lim_{ n \to \infty }a_{n}=\pm \infty=\lim_{ n \to \infty }b_{n}$$$\implies \lim_{ n \to \infty }(a_{n}+b_{n})=\pm \infty$$
> 3. Se $\lim_{ n \to \infty }a_{n}=a\in \mathbb{R} \setminus \{0\}, \lim_{ n \to \infty }b_{n}=\pm \infty$$$\implies \lim_{ n \to \infty }(a_{n}\cdot b_{n}) = \pm \infty$$
> 4. Se $\lim_{ n \to \infty }a_{n}=a\in \mathbb{R}\setminus \{ 0 \} \land \lim_{ n \to \infty }b_{n}=0^{\pm}$$$\implies\lim_{ n \to \infty } \frac{a_{n}}{b_{n}}=\pm \infty$$
> 5. Se $\lim_{ n \to \infty }a_{n}=a\in \mathbb{R}\land \lim_{ n \to \infty }b_{n}=\pm \infty$$$\implies \lim_{ n \to \infty } \frac{a_{n}}{b_{n}}=0$$
> 6. Se $\lim_{ n \to \infty }a_{n}=a\in \mathbb{R} \land \lim_{ n \to \infty }b_{n}=b \in \mathbb{R}$$$\implies \lim_{ n \to \infty } a_{n}^{b_{n}}=\lim_{ n \to \infty } e^{b_{n}\ln a_{n}}$$
> 7. Se $\lim_{ n \to \infty }a_{n}\in \mathbb{R}, a\geq 0$ e $\lim_{ n \to \infty }b_{n}\in \mathbb{R}$$$\implies\lim_{ n \to \infty }a_{n}^{b_{n}}=a^b$$
> dove:
> $$a^{+\infty}=+\infty\ \forall\ a>1$$
> $$a^{-\infty}=0      \ \forall\ a>1$$
> $$a^{+\infty}=0      \ \forall\ a>1$$
> $$a^{-\infty}=+\infty\ \forall\ a\in[0,1)$$
> $$+\infty^b = +\infty\ \forall\ b>0$$
> $$+\infty^b = 0      \ \forall\ b<0$$

> [!info] Osservazione
> Rimangono escluse le seguenti forme di indeterminazioni:
> $$\infty-\infty,\ \infty \cdot 0,\ \frac{\infty}{\infty},\ \frac{0}{0}$$
