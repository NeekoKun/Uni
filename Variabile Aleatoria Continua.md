#IeS 
> [!warning] Definizione
> $X$ è una [[Variabile Aleatoria Continua]], $X : \Omega \to \mathbb{R}$, se le probabilità possono essere calcolate come integrali definiti di una funzione $f_{X}$ detta **Densità di Probabilità**
> $$P(a \leq X \leq b) = P(a < X < b)$$

## Calcolo
- [[Valore Atteso#Valore Atteso per Variabile Aleatoria continua]]
- [[Varianza]]
## Proprietà di $f_{X}$
- $f_{X}(x) \geq 0\ \forall\ x \in \mathbb{R}$
- $1 = P(\Omega) = \int_{-\infty}^\infty f_{X}(x) = P(X = \mathbb{R})$
