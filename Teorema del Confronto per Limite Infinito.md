#analisi 
> [!error] Teorema
> Se $x_{0} \in \overline{\mathbb{R}},\ f,g : I \subseteq \mathbb{R} \to \mathbb{R}, x_{0}$ punto di accumulazione di $I$ e definitivamente per $x \to x_{0}$ $$f(x) \leq g(x)$$
> allora:
> 1. $\lim_{  x \to x_{0} } f(x) = +\infty \implies \lim_{ x \to x_{0} }g(x) = +\infty$
> 2. $\lim_{  x \to x_{0} } g(x) = -\infty \implies \lim_{ x \to x_{0} }f(x) = -\infty$

## Corollari
> [!error] Corollario
> Se $\lim_{ x \to x_{0} } g(x) = 0, x_{0} \in \overline{\mathbb{R}}$ e se $|f(x)| < g(x)$ allora $\lim_{ x \to x_{0} } = 0$

> [!error] Corollario
> Se $\lim_{ x \to x_{0} } g(x) = 0, x_{0} \in \overline{\mathbb{R}}$ e se $f(x)$ è limitata definitivamente per $x \to x_{0}$, allora $$\exists \lim_{ x \to x_{0} } f(x) - g(x) = 0 $$
